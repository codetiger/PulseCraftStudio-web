# Coordinate System

- Y-up, right-handed coordinate system
- Units: meters
- Scene center: (0, 0, 0)
- Colors: hex format #RRGGBB

## Rotations (Euler Angles in Radians)

| Degrees | Radians |
|---------|---------|
| 0° | 0 |
| 30° | 0.5236 |
| 45° | 0.7854 |
| 60° | 1.0472 |
| 90° | 1.5708 |
| 120° | 2.0944 |
| 180° | 3.1416 |
| 270° | 4.7124 |
| 360° | 6.2832 |

Rotation order: XYZ (Tait-Bryan). Positive rotation follows right-hand rule.
